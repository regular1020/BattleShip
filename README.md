# BattleShip

Battleship Board Game

# 개요

배틀쉽 보드게임입니다.

플레이어는 한 턴마다 공격을 선언합니다.

공격의 선언은 행/열의 번호를 입력하여 선언 할 수 있습니다.(ex: A 3)

공격이 상대방의 배에 맞으면 HIT, 맞지 않으면 MISS로 알려줍니다.

상대방의 모든 배를 침몰시키면 승리입니다.

상대방은 컴퓨터가 알고리즘에 따라 자동으로 진행합니다.
